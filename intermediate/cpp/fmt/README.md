# Readme

This is an example that demonstrates how {fmt} can be used with Bazel.

## How to run

Install [Bazel](https://docs.bazel.build/versions/4.2.1/install.html) and run:


```shell
bazel run //:Demo
```

This should output `The answer is 42`.
